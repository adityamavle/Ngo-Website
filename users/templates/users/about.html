ABOUT US

Aim of our project is to build a platform like a directory to make the search for NGOs easier. It will help to establish a connection between NGOs and the enthusiast users who would like to volunteer to work or provide donations to help benefit the good cause carried out by different NGOs.

Objectives of this project is to generate a systematic, user -friendly search engine website for the users and to bring all small and big NGO onto the same platform.
